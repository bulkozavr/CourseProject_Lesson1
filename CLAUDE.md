# Javarush Project Rules

## Tech Stack
- Single `index.html` file — no npm, no backend, no build tools
- Pure HTML + CSS + vanilla JS only

## Posters / Visual Assets
- All posters and graphic elements are drawn with **CSS gradients** (linear-gradient, radial-gradient, conic-gradient, etc.)
- No images loaded from disk (no `<img src="...">`, no `url(...)` pointing to local files)
- SVG is allowed, but must be inline in the HTML

## Theme
- **Dark theme** as the default and only color scheme
- **Blue accents** for interactive elements, highlights, and key visual elements (#3b82f6 or similar blue tones)
- Use dark background colors (e.g. #0f172a, #1e293b, #111827) with light text (#e2e8f0, #f8fafc)

## General
- Mobile-responsive layout
- Clean, minimal code
